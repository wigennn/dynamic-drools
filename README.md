# dynamic-drools
动态更新规则的drools规则引擎demo

## 背景
```
遇到一个业务场景，现在需要对于审批的发票做各种规则校验，
对于提交审批的发票，符合规则的则自动审批通过，不符合的驳回。
通常对于发票字段的比较就是一个一个字段的去与定义的规则比较，繁琐，代码不高级，
了解到规则引擎，可以把比较的过程交给规则引擎执行，只需要定义相应的规则语句。
想了解下drools的使用，就简单写了下，整体代码框架搭好
```

## 主要使用技术栈
- drools (规则引擎)
- freemarker (模板引擎)
- mysql
